# Installation

## Stable release

To install BinaryOrNot, run this command in your terminal:

```sh
uv add binaryornot
```

Or if you prefer to use `pip`:

```sh
pip install binaryornot
```

## From source

The source files for BinaryOrNot can be downloaded from the [Github repo](https://github.com/binaryornot/binaryornot).

You can either clone the public repository:

```sh
git clone https://github.com/binaryornot/binaryornot
```

Or download the [tarball](https://github.com/binaryornot/binaryornot/tarball/main):

```sh
curl -OJL https://github.com/binaryornot/binaryornot/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd binaryornot
uv sync
```
